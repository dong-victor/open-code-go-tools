配置与密钥管理

保存密钥将直接同步写入本地的 `config.json` 配置文件。

Claude Code 环境配置 

$env:ANTHROPIC_BASE_URL = "http://127.0.0.1:8787"
$env:ANTHROPIC_API_KEY = "ocgt-local-proxy"
$env:ANTHROPIC_CUSTOM_HEADERS = "X-Ocgt-Profile: opencode-go"
$env:CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS = "1"
$env:CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY = "1"
$env:ANTHROPIC_MODEL = "kimi-k2.6"

在你的开发终端中执行上述配置命令，然后运行 `claude` 即可通过本地代理发起请求。

CC Switch 提供商配置 

{
  "name": "ocgt-opencode-go",
  "type": "anthropic",
  "baseURL": "http://127.0.0.1:8787",
  "apiKey": "ocgt-local-proxy",
  "model": "kimi-k2.6",
  "headers": {
    "X-Ocgt-Profile": "opencode-go"
  }
}

你可以将以上 JSON 代码段导入到 CC Switch 客户端的 Provider 配置中。
